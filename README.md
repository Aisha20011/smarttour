# SmartTour - دليلك السياحي الذكي
## SmartTour - Your Smart Tourist Guide

<div dir="rtl">

## 📱 نظرة عامة

SmartTour هو تطبيق جوال ذكي مصمم ليكون دليلاً سياحياً شاملاً يساعد المستخدمين على استكشاف المدن والمواقع السياحية بطريقة تفاعلية ومبتكرة.

## ✨ الميزات الرئيسية

- 🔐 نظام مصادقة متكامل (Email, Google)
- 🗺️ خرائط تفاعلية مع عرض الأماكن القريبة
- 📅 تخطيط وإدارة الرحلات
- 🤖 توصيات ذكية مخصصة
- 📍 تفاصيل شاملة عن الأماكن السياحية
- 🌐 دعم متعدد اللغات (العربية والإنجليزية)
- 🎨 واجهة مستخدم عصرية وسهلة الاستخدام

## 🛠️ التقنيات المستخدمة

- **Flutter** - إطار عمل متعدد المنصات
- **Firebase** - المصادقة والتخزين
- **Provider** - إدارة الحالة
- **flutter_map** - الخرائط التفاعلية
- **geolocator** - خدمات الموقع

## 🚀 البدء السريع

### المتطلبات

- Flutter SDK 3.0+
- Dart SDK 3.0+
- Android Studio / VS Code
- حساب Firebase

### التثبيت

```bash
# استنساخ المشروع
git clone <repository-url>
cd smarttour

# تثبيت التبعيات
flutter pub get

# تشغيل التطبيق
flutter run
```

## 📱 Overview

SmartTour is a smart mobile application designed to be a comprehensive tourist guide that helps users explore cities and tourist sites in an interactive and innovative way.

## ✨ Key Features

- 🔐 Integrated Authentication System (Email, Google)
- 🗺️ Interactive Maps with Nearby Places
- 📅 Trip Planning and Management
- 🤖 Smart Personalized Recommendations
- 📍 Comprehensive Place Details
- 🌐 Multi-language Support (Arabic & English)
- 🎨 Modern and User-Friendly Interface

## 🛠️ Technologies Used

- **Flutter** - Cross-platform framework
- **Firebase** - Authentication & Storage
- **Provider** - State Management
- **flutter_map** - Interactive Maps
- **geolocator** - Location Services

## 🚀 Quick Start

### Requirements

- Flutter SDK 3.0+
- Dart SDK 3.0+
- Android Studio / VS Code
- Firebase Account

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd smarttour

# Install dependencies
flutter pub get

# Run the app
flutter run
```
